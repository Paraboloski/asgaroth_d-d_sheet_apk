# Asgaroth D&D Character Sheet

Applicazione desktop standalone per la gestione della scheda del personaggio di Dungeons & Dragons 5e. Progettata per essere rapida, parzialmente automatizzata e utilizzabile completamente offline.

## Installazione

1. Naviga nella sezione **Releases** sul lato destro di questa repository.
2. Scarica l'ultima versione del file eseguibile (`.exe`).
3. Avvia il file tramite doppio clic. Non è richiesta alcuna installazione complessa o connessione internet.

## Funzionalità e Utilizzo

* **Calcolo Automatico:** Modifica il punteggio base delle caratteristiche (es. 16). Il sistema calcolerà e aggiornerà automaticamente il relativo modificatore (es. +3) e l'iniziativa.
* **Competenze:** Spuntando la casella accanto a un Tiro Salvezza o a un'Abilità, il sistema sommerà in automatico il Bonus di Competenza al modificatore della statistica associata. Modificando il Bonus di Competenza, tutti i valori spuntati si aggiorneranno di conseguenza.
* **Tabelle Dinamiche:** Le sezioni relative ad Attacchi, Tratti di Classe e Profilo Psicologico permettono di aggiungere o rimuovere righe tramite gli appositi pulsanti (+ e -).
* **Stampa:** Il pulsante "Stampa PDF" formatta l'interfaccia rimuovendo i pulsanti di controllo per un'esportazione pulita e pronta per la stampa cartacea.

## Gestione dei Salvataggi

Per salvare le modifiche apportate alla scheda, è sufficiente cliccare sul pulsante **Salva Modifiche** situato nella barra di navigazione superiore. 

L'applicazione salva i dati direttamente nella memoria locale del sistema operativo. Non verranno generati o scaricati file esterni. È possibile chiudere l'applicazione in totale sicurezza: al successivo avvio, i dati verranno caricati automaticamente ripristinando lo stato esatto dell'ultima sessione.

Per svuotare la scheda e crearne una nuova, utilizzare il pulsante **Resetta Scheda**. Questa azione eliminerà il salvataggio locale.

# Asgaroth D&D Character Sheet (English)

A standalone desktop application for managing Dungeons & Dragons 5e character sheets. Designed to be fast, partially automated, and fully offline.

## Installation

1. Navigate to the **Releases** section on the right side of this repository.
2. Download the latest version of the executable file (`.exe`).
3. Double-click the file to run the application. No complex installation or internet connection is required.

## Features and Usage

* **Automated Calculations:** Edit the base ability scores (e.g., 16). The system will automatically calculate and update the corresponding modifier (e.g., +3) and initiative.
* **Proficiencies:** By checking the box next to a Saving Throw or Skill, the system will automatically add your Proficiency Bonus to the associated ability modifier. If you update your Proficiency Bonus, all checked values will update accordingly.
* **Dynamic Tables:** The Attacks, Class Features, and Psychological Profile sections allow you to add or remove rows using the provided buttons (+ and -).
* **Print:** The "Stampa PDF" (Print PDF) button formats the interface by hiding the control buttons, providing a clean export ready for physical printing.

## Save System

To save your progress, click the **Salva Modifiche** (Save) button located in the top navigation bar.

The application saves your data directly to your operating system's local storage. No external files will be generated or downloaded. You can safely close the application: upon the next launch, your data will load automatically, restoring the exact state of your last session.

To clear the sheet and start a new one, use the **Resetta Scheda** (Reset) button. Warning: this action will permanently delete your local save.
